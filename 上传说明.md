# 公开发布说明

这个文件夹可以直接用于公开发布，里面只保留了网站需要的文件：

- `index.html`
- `styles.css`
- `script.js`
- `avatar.svg`

## GitHub Pages

1. 登录 GitHub。
2. 新建一个公开仓库。
3. 把这个文件夹里的文件上传到仓库根目录。
4. 打开仓库 `Settings -> Pages`。
5. 选择 `Deploy from a branch`。
6. 分支选择 `main`，目录选择 `/(root)`。
7. 保存后等待几分钟，网页就会公开可访问。

## Cloudflare Pages

1. 登录 Cloudflare。
2. 打开 `Workers & Pages`。
3. 选择创建项目并上传这个文件夹里的文件。
4. 发布后会得到一个公开网址。

## 上线前记得改

1. 在 `index.html` 里改成你的名字。
2. 改邮箱地址。
3. 改 GitHub 和 LinkedIn 链接。
4. 改项目内容。
